# ATrack

ATrack is designed to allow users to publicly grade albums, songs, and singles based on their personal enjoyment of it. Users may add comments to specific timestamps in the song allowing them to discuss their opinions about specific parts of a song.

## Technology Stack
- HTML, CSS, JavaScript
- Node.js, Express
- Docker, PostgreSQL

## Prerequisites
- Docker
- Node.js
